# BuildAForm

A Mini-Typeform clone with AI-powered form generation.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS & Shadcn UI
- **ORM**: Prisma
- **Database**: Supabase / PostgreSQL
- **Validation**: Zod
- **AI**: Ollama (Local)

## Development Workflow
We use Gitflow. All features should be developed in `feature/` branches and merged into `develop`.
Progress is tracked via GitHub Issues.
