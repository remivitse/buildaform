# BuildAForm

A Mini-Typeform clone with AI-powered form generation.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4 & Shadcn UI
- **ORM**: Prisma
- **Database**: Supabase / PostgreSQL
- **Validation**: Zod
- **AI**: Ollama (Local)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development Workflow
We use Gitflow. All features should be developed in `feature/` branches and merged into `develop`.
Progress is tracked via GitHub Issues.
PRs should follow the Conventional Comments guidelines.
